# Task 4: Classification with Logistic Regression

## Objective

Build a binary classification model using Logistic Regression and evaluate its performance using multiple classification metrics.

---

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

---

## Dataset

Breast Cancer Wisconsin Dataset from Scikit-Learn.

Target Classes:

* Malignant
* Benign

---

## Steps Performed

### 1. Data Loading

Loaded the Breast Cancer Wisconsin Dataset.

### 2. Train-Test Split

Split dataset into:

* 80% Training Data
* 20% Testing Data

### 3. Feature Scaling

Applied StandardScaler to standardize features.

### 4. Model Training

Trained Logistic Regression model using Scikit-Learn.

### 5. Prediction

Generated predictions and prediction probabilities.

### 6. Evaluation Metrics

Evaluated using:

* Accuracy
* Precision
* Recall
* ROC-AUC Score
* Confusion Matrix

### 7. ROC Curve

Visualized classifier performance using ROC Curve.

### 8. Threshold Tuning

Tested different probability thresholds and compared Accuracy, Precision, and Recall.

### 9. Sigmoid Function

Visualized the Sigmoid Curve used in Logistic Regression.
