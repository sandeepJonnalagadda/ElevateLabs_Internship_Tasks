# Task 6: K-Nearest Neighbors (KNN) Classification

## Objective

Implement the K-Nearest Neighbors (KNN) algorithm for classification and understand the impact of feature scaling and K-value selection.

---

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

---

## Dataset

Iris Dataset from Scikit-Learn.

Target Classes:

* Setosa
* Versicolor
* Virginica

---

## Steps Performed

### 1. Data Loading

Loaded the Iris Dataset.

### 2. Feature Scaling

Applied StandardScaler to normalize features.

### 3. Train-Test Split

Split data into training and testing sets.

### 4. K Selection

Tested K values from 1 to 20 and selected the best K based on accuracy.

### 5. Model Training

Trained a KNeighborsClassifier using the optimal K value.

### 6. Model Evaluation

Evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report

### 7. Decision Boundary Visualization

Visualized KNN classification boundaries using the first two features.
