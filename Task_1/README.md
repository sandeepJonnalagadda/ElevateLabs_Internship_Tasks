# Task 1: Data Cleaning & Preprocessing

## Objective
Clean and preprocess raw Titanic dataset for Machine Learning.

## Steps Performed

1. Imported dataset using Pandas
2. Checked dataset information and missing values
3. Handled missing values
4. Encoded categorical features
5. Standardized numerical features
6. Detected outliers using boxplots
7. Removed outliers using IQR method
8. Saved cleaned dataset

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Output

- Cleaned Titanic Dataset
- Boxplot Before Outlier Removal
- Boxplot After Outlier Removal
