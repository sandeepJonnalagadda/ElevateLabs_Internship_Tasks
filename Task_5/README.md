# Task 5: Decision Trees and Random Forests

## Objective

Learn tree-based machine learning models for classification and compare the performance of Decision Trees and Random Forests.

---

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

---

## Dataset

Breast Cancer Wisconsin Dataset from Scikit-Learn.

Target Classes:

* Malignant
* Benign

---

## Steps Performed

### 1. Data Loading

Loaded the Breast Cancer Dataset.

### 2. Train-Test Split

Split the dataset into training and testing sets.

### 3. Decision Tree Classifier

* Trained a Decision Tree Classifier.
* Controlled overfitting using `max_depth`.
* Visualized the decision tree.

### 4. Random Forest Classifier

* Trained a Random Forest model.
* Compared performance with Decision Tree.

### 5. Feature Importance

* Extracted feature importances.
* Visualized top contributing features.

### 6. Cross Validation

Performed 5-Fold Cross Validation to evaluate model stability.

### 7. Model Comparison

Compared Decision Tree and Random Forest based on accuracy.

---

## Results

* Random Forest generally achieved higher accuracy than a single Decision Tree.
* Feature importance helped identify the most influential predictors.
* Cross-validation provided a more reliable estimate of model performance.
